# Data Types

A variable is eather an object or a primitive data type

**Primitive Data types**
**1. Number**
    No int or double just numbers.
**2. String**
**3. Boolean**
**4. Undefined**
    Value of a variable is not yet defined
**5. Null**
    Similar to Undiefined but different
**6. Symbol**
**7. BigInt**

**Javascript has dynamic typing**
    data types are determined automatically
    The VALUE has a type, not the VARIABLE
        Pro - Easy to use
        Con - Harder to debug
