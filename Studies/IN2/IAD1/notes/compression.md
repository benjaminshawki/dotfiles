# LZW

LZW is a dictionary-based compression algorithm. It is similar to Huffman coding in that it uses a dictionary to map symbols to codes. However, LZW uses a sliding window to build the dictionary, and it uses a fixed-width code length.

# Huffman

Huffman coding is a dictionary-based compression algorithm. It uses a variable-width code length to map symbols to codes. The algorithm builds a binary tree to represent the code length of each symbol, and it uses the tree to encode the input data.
