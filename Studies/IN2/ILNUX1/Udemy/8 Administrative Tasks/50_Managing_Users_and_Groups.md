>`useradd` 
>
>`-d` create user directory 
>
>`-m` create additional directory
>
>`-s` shell-type ie `/bin/tsch/` `/bin/sh`
>
>`-c` User name
>
>`-G` associate multiple groups to a user

>`userdel`
>
>`-r` delete home directory of user
>
>`-f` force even while logged in

>`usermod`
> 
> `-d` change directory
> 
> `-c` change name
> 
> `-L` Lock user
> 
> `-U` unlock user
> 
> `-aG` add groups

`passwd` to change password
