2) Waarom is de requirements analyse bij ieder project zo belangrijk? ( denk aan de 
PDCA ) 
PDCA staat voor Plan, Do, Check, Act. Dit is een cyclus die ervoor zorgt dat je een project goed afrondt. De requirements analyse is de eerste stap in deze cyclus. Als je deze stap niet goed doet, dan kan het zijn dat je project niet goed wordt afgerond.

3) Wat betekent ISMS? 
ISMS staat voor Information Security Management System. Dit is een systeem dat ervoor zorgt dat de informatiebeveiliging binnen een organisatie goed geregeld is.
ISMS is een manier om de informatiebeveiliging binnen een organisatie te managen. Het is een managementsysteem dat ervoor zorgt dat de informatiebeveiliging binnen een organisatie goed geregeld is. Het is van belang dat je dit systeem goed inricht, zodat je de risico's van de informatiebeveiliging goed kunt managen.

5) Wat is een threat? En waarmee wordt deze altijd vergezeld? 
Een threat is een gebeurtenis die schade kan aanrichten aan een organisatie. Deze wordt altijd vergezeld met een de kans. De kans geeft aan hoe groot de kans is dat de threat daadwerkelijk plaatsvindt.
 
6) Moet een threat altijd worden afgedekt? 
Nee, een threat hoeft niet altijd te worden afgedekt. Het is van belang dat je kijkt naar de impact en de kans van een threat. Als de impact en de kans laag zijn, dan hoef je deze niet perse af te dekken.
 
7) Wat is defense in depth?
Defense in depth is een beveiligingsstrategie die ervoor zorgt dat je meerdere lagen van beveiliging hebt. Dit zorgt ervoor dat als een laag wordt doorbroken, er nog een andere laag is die de aanval kan tegenhouden.
 
8) Hoe kun je onderscheid tussen gebruikers afdwingen? Hoe doet windows dit? Hoe 
doet linux dit? 
Je kunt onderscheid tussen gebruikers afdwingen door middel van authenticatie en autorisatie. Windows doet dit door middel van Active Directory. Linux doet dit door middel van de /etc/passwd en /etc/shadow bestanden.
Linux heeft ook een bestand /etc/group waarin je groepen kunt aanmaken. Dit is handig als je meerdere gebruikers dezelfde rechten wilt geven.
 
9) Wat is een firewall? 
Een firewall is een systeem dat ervoor zorgt dat ongewenst verkeer wordt tegengehouden. Dit kan zowel softwarematig als hardwarematig zijn.
 
10) Leg uit wat IDS doet en wat IPS doet?  
Een IDS (Intrusion Detection System) is een systeem dat ervoor zorgt dat je inzicht krijgt in wat er gebeurt op je netwerk. Een IPS (Intrusion Prevention System) is een systeem dat ervoor zorgt dat je actie kunt ondernemen als er iets verdachts gebeurt op je netwerk.
 
11) Hoe kun je de gebruikers ( end-point ) beveiligen? 
Je kunt de gebruikers beveiligen door middel van antivirussoftware, firewalls en encryptie. Het is ook van belang dat je de gebruikers bewust maakt van de risico's van het internet.
